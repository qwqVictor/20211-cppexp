# 综合实验 1：Josephus 问题

本代码以 Unlicense 开源许可证开放源代码。

作者：黄凯升 <2020215138@stu.cqupt.edu.cn>

### 实现功能

- 支持从第 k 个小孩开始数
- 从标准输入流读取假定的 m 和 k
- 获得胜利者，并输出编号和姓名

### 输入格式

一行三个整数 n, m, k，分别表示小孩数，每几个小孩离开，从第几个小孩开始数。

然后 n 行，每行两个用空格分隔的字符串，表示学号和姓名。

在文件 `__gitignored_data_input.txt` 中，有按照课程要求的包含本班人员学号和姓名的输入样例，样例解释见 docx 文档。
